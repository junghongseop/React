import구문이 사용됨 = 특정 파일을 불러오는 것
리엑트로 만든 프로젝트의 자바스크립트 파일에서는 import를 사용하여 다른 파일들을 불러와 사용할 수 있음

모듈을 불어와서 사용하는 것은 브라우저에는 없던 기능임
브라우저가 아닌환경에서 자바스크립트를 실핼 할 수 있게 해주는 환경인 Node.js에서 지원하는 기능

브라우저에서도 사용하기 위해 번들러를 사용함
번들=파일을 묶듯이 연결하는 것

대표적인 번들러는 웹팩, Parcel, browserify
요즘은 주로 웹팩을 사용하는 추세, 편의성과 확장성이 다른 도구보다 뛰어 나기때문임
번들러 도구를 사용하면 import로 모듈을 불러왔을 때 불러온 모듈은 모두 합쳐서 하나의 파일을 생성

최적화 과정에서 여러개의 파일로 분리될 수도 있음

웹팩을 사용하면 svg파일과 css파일도 불어와서 사용 가능
파일들을 불러오는 것은 웹팩의 로더라는 기능이 담당
로더는 css파일을 불러올 수 있게 해주고, 웹 폰트나 미디어 파일등을 불러올 수 있게 해줌
그리고 자바스크립트 파일들을 불러오면서 최신 자바스크립트 문법으로 작성된 코드를 바벨이라는 도구를 사용하여 ES5문법으로 변환해줌

웹팩의 로더는 원래 직접 설치하고 설정해야 하지만 리액트 프로젝트를 만들때 사용했던 create-react-app이 번거로운 작업을 모두 대신 해주기 때문에 별도의 설정을 할 필요가 없음

import logo from './logo.svg';

이 코드는 App이라는 컴포넌트를 만들어줌
function키워드를 사용하여 컴포넌트를 만들면 함수 컴포넌트라고 부름
프로젝트에서 컴포넌트를 렌더링하면 함수에서 반환하고 있는 내용을 나타냄

함수에서 반환하는 내용을 보면 HTML을 작성한거 같은데 이는 JSX라는 코드이다
