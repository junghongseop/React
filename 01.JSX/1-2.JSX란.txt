JSX는 자바스크립트의 확장 문법이며 XML과 매수 비슷하게 생김
브라우저에서 실행되기 전에 코드가 번들링되는 과정에서 바벨을 사용하여 일반 자바스크립트 형태의 코드로 변환

function App() {

  return (
    <div>
      Hello <b>react</b>
    </div>    
  );
}

↓↓↓↓↓↓↓↓↓↓↓↓↓↓

function App() {
  return React.createElement("div", null, "Hello", React.createElement("b", null, "React"));
}

랜더링할 때마다 JSX코드를 작성하는 것이 아니라 매번 React.creatElement함수를 사용해야 한다면 매우 불편
JSX를 사용하면 매우 편하게 UI를 렌더링할수 있음
